`@wallet-ui/core` is the framework-agnostic foundation of Wallet UI. It provides the necessary building blocks for state management and configuration that power the UI components and hooks in the React package.

You will usually touch this package indirectly through `@wallet-ui/react`, but the core docs are useful when you need to customize cluster support, storage behavior, or lower-level utilities.

## Explore the core library

- [**Clusters**](./clusters): Learn how to configure the Solana clusters your app supports.
- [**Storage**](./storage): Understand how to manage and persist state.
- [**Utilities**](./utilities): Discover the available utility functions.
